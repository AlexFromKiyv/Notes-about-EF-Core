# Послідовності

Послідовності – це функція, яка зазвичай підтримується лише реляційними базами даних. Якщо ви використовуєте нереляційну базу даних, таку як Azure Cosmos DB, перевірте документацію до бази даних щодо створення унікальних значень.

Послідовність генерує унікальні, послідовні числові значення в базі даних. Послідовності не пов'язані з певною таблицею, і можна налаштувати кілька таблиць для отримання значень з однієї й тієї ж послідовності.

## Базове використання

Ви можете налаштувати послідовність у моделі, а потім використовувати її для створення значень властивостей:

```cs
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.HasSequence<int>("OrderNumbers");

    modelBuilder.Entity<Order>()
        .Property(o => o.OrderNo)
        .HasDefaultValueSql("NEXT VALUE FOR OrderNumbers");
}
```
Зверніть увагу, що конкретний SQL-запит, який використовується для генерації значення з послідовності, залежить від бази даних; наведений вище приклад працює на SQL Server, але не працюватиме на інших базах даних. Для отримання додаткової інформації зверніться до документації вашої конкретної бази даних.

## Налаштування параметрів послідовності

Ви також можете налаштувати додаткові аспекти послідовності, такі як її схема, початкове значення, приріст тощо:

```cs
protected override void OnModelCreating(ModelBuilder modelBuilder)
{
    modelBuilder.HasSequence<int>("OrderNumbers", schema: "shared")
        .StartsAt(1000)
        .IncrementsBy(5);
}
```
